# noted
Simple Notepad Demo for [Ionic](http://ionicframework.com/) 2.

###Installation

git clone https://github.com/ntancardoso/noted.git

cd noted

npm install


### Run Unit Tests

npm test


### Start

ionic serve


## Dependencies

* **@Angular:** 2.0.0-rc.1
* **Ionic:** 2.0.0-beta.7
